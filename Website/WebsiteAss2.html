To install your first mod, follow these simple steps:

  1. Download your desired mod
  2. Locate your game installation path
  3. Open an existing or create a mods folder
  4. Extract the mod files into the mods folder
  5. Complete any other instructions provided by the mod author
  6. Launch the game
