Popular mods for The Witcher 3

1. The Witcher 3 HD Reworked Project

This is a mod by Halk Hogan PL, which aims to improve the game's graprics by reworking models and textures to better quality, while preserving the original art style. You can download it here.

2. Over 9000 - Weight limit mod

A simple, yet useful mod for players, who feel annoyed by Geralt being constantly encumbered by heaps of items. This mod changes the weight limit to over 9000, so this issue will trouble you no more! Get it here.

3. Fast travel from anywhere

Again, not a very comlicated mod, but quite a useful one. It eliminates the problem some players might have with the fast travel system ingame, which allows fast travel only at signposts. After installation, you can travel to any signpost from any location. Download it over here.

4. Super Turbo Lighting Mod

Changes the ligting in all zones of the game to a previous version considered better by many. View screenshots to see if you would like to download it or not. And of course if you do, get it here.

5. TradeMan

Tired of merchants setting high prices for goods? Want to sell your items to any merchant and not worry about them affording it? Then this mod is for you. You can now edit every aspect of trading to your liking! Download here.
