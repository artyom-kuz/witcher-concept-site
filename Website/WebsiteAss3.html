Script Merger is an application created by AnotherSymbiote on nexusmods.com, which allows the user to dissolve conflicts between modded scripts within The Witcher 3.

This application is used when two or more mods cannot work together because of using different versions of edited files from the game.

Wghost81 on YouTube does a good job of explaining how it works in this short video.
